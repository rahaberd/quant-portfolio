# Math Tools

A command-line calculator for common combinatorics operations.

## Features

- **Factorial** — calculates n!
- **Combination** — calculates C(n, k), the number of ways to choose k items from n
- **Word Permutations** — calculates the number of unique arrangements of a word's letters

## Usage

```
python math_tools.py
```

You'll be presented with a menu to select a calculation. Follow the prompts to enter your values.

## Requirements

Python 3.x (no external dependencies)
